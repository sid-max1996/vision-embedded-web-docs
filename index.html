# Введение

Cubic Vision Embedded Web - это библиотека для взаимодействия встраивоемого веб контента с плеером. Раздел Интерфейс плеера содержит описание интерфейса, который должен быть реализован на плеерах. Раздел "Методы веб контента" описывает методы для создания встраивоемого в плееры веб контента. На данный момент встраиваемым веб контеном являются: Web Widgets на Андроид и виджет Web Project.
